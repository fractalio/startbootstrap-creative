Data Integrity & Durability

While the cloud has abstracted away some of the hardware dependency, it has come at the cost of security and privacy, and yet, does not deliver true hardware-decoupled data durability.

    UNICell is a high-performance, feature-rich & affordable unified storage platform for a wide range of business applications.

    With block & file access in a single appliance, as well as support for a wide range of services and protocols, UNICell is an affordable and versatile data storage platform.

    Whether as central storage for file sharing, off-site replication storage for business continuity, backup repository for mission-critical VMs, shared storage for video editing or central storage for video surveillance, UNICell is a reliable, fast, and easy-to-manage unified storage solution.

    Unicell

    The UNICell consists of a storage controller and only as many disks as necessary, to ensure a well balanced controller:spindle:network bandwidth, packaged into a compact, energy-efficient rack enclosure.

    Rightsized IO pipelines within each UNICell are a key design element; eliminating internal bottlenecks within each UNICell ensures full IO and system bandwidth of each UNICell is available to the end application

    Upgradeability to GRIDCell ensures ability to scale the infrastructure for performance or capacity without forklift upgrades and ensures investment protection.

    Features

    - Unified file & block storage
    - Single dashboard webUI
    - RAID protection against disk failures
    - ZFS for on-disk data integrity
    - Snapshots
    - Deduplication & compression
    - Ugradable to grid storage

    Applications

    - File storage & sharing
    - Video surveillance
    - Data archival
    - Media & entertainment storage
    - VM repositories

    IntegralStor GridCell is a scale-out grid-storage framework for the delivery of data storage and information management functions as a layered set of scalable, fault-resilient and highly-available services spanning in-premise infrastructure to the cloud.

    The IntegralStor architecture, based on the latest generation energy-efficient hardware and open-source distributed filesystems delivers high performance file access, at a fraction of the cost of proprietary high-performance storage.

    GRIDCells

    The autonomous, control & data storage elements that form the hardware building-blocks of IntegralStor.

    Each GRIDCell consists of a storage controller and as few disks as are necessary, to ensure a well balanced controller:spindle:network bandwidth, packaged into a compact, energy-efficient rack enclosure.

    Rightsized IO pipelines within each GRIDCell are a key design element; eliminating internal bottlenecks within each GRIDCell ensures full IO and system bandwidth of each GRIDCell is available to the end application, and consequently therefore, the aggregate bandwidth of all GRIDCells.

    Features

    - Global namespace
    - Snapshots
    - Replication
    - ZFS on-disk
    - Single dashboard webUI
    - Small, modular, building-blocks

    Applications

    - Media & entertainment
    - Video surveillance
    - HPC storage
    - Biosciences
    - Data Archival
    - VM repositories
    - High-performance / high-capacity file access
